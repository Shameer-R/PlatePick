# **App Name**: PlatePick

## Core Features:

- User Authentication: Secure user login and registration with email and password.
- Meal Plan Request: Users submit requests for meal plans specifying dietary restrictions, cuisine preferences, and number of meals.
- AI Meal Planner: Leverage Gemini to generate meal plans based on user requests, incorporating nutritional balance and variety.  The LLM acts as a tool to provide creative solutions to dietary constraints and other information supplied by the user.
- Meal Plan Display: Present the generated meal plan in a clear and organized format, including meal names, recipes, and nutritional information.
- Meal Plan Storage: Store user meal plans in Firestore for future access.
- User Profile: Allow users to manage their profile and meal preferences.

## Style Guidelines:

- Primary color: Forest green (#3F704F), evoking health, nature, and growth related to balanced nutrition.
- Background color: Very light, desaturated green (#F0F4EF), providing a calm, clean backdrop that enhances the readability of text and visual elements.
- Accent color: Mustard Yellow (#D6AE4A) providing contrast to the primarily green interface.
- Headline font: 'Belleza' sans-serif, chosen for its stylish personality, best for headlines and short blocks of text. Body font: 'Alegreya' serif, chosen for its legibility and contemporary elegance.
- Use clean and modern icons to represent different food categories and dietary restrictions.
- Employ a card-based layout to present meal plans and recipes in an organized manner.
- Use subtle transition animations to create a smooth user experience.
